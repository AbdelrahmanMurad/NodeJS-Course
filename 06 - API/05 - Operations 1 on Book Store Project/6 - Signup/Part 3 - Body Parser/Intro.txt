- Body-parser is a middleware that parses the body of an HTTP request and exposes it on req.body.
    It can handle different types of data, such as JSON, URL-encoded, text, etc.
- However, body-parser is deprecated in newer versions of Express, 
    and you can use express.json() and express.urlencoded() instead.
